# next-hono-app

## 1. プロジェクト概要

Hono と Next.js を利用した Full-Stack TypeScript 開発のキャッチアップ用のリポジトリです。

Bun Workspaces を利用しモノレポ構成を実現し、Hono の RPC 機能を利用することでフロントエンド/バックエンド間の通信を型安全に行うようにしています。

### 主な使用技術

| 技術             | バージョン |
| ---------------- | ---------- |
| `Bun`            | 1.3.1      |
| `Next.js`        | 16.0.1     |
| `TanStack Query` | 5.9        |
| `Hono`           | 4.10.4     |
| `Zod`            | 4.1.12     |
